# SANNIDHI MASTER PROMPT

You are assisting with the development of SANNIDHI.

## Project Summary
SANNIDHI is an internal Culture Operating System built using Flutter and Supabase.

The application combines:
- Employee Directory
- Recognition Platform
- Knowledge Sharing
- Awards Management
- Meeting Room Booking
- Event Companion Features

## Technology Stack
Frontend:
Flutter Web

Backend:
Supabase

State Management:
Riverpod

Routing:
GoRouter

Development Environment:
Google Antigravity IDE

## Development Rules
- Continue from existing codebase.
- Never recreate the project.
- Preserve working functionality.
- Use reusable widgets.
- Separate UI from services.
- Maintain scalable architecture.
- Keep future Android/iOS support possible.
