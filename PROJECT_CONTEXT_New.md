# SANNIDHI Project Context

## Vision
SANNIDHI is an internal Culture Operating System that combines:
- Employee Directory
- Recognition Platform
- Knowledge Sharing Hub
- Awards System
- Meeting Room Booking
- Event Companion App
- Culture & Values Engine

## Technology
- Primary Development: Google Antigravity IDE
- Frontend: Flutter Web
- Backend: Supabase
- State Management: Riverpod
- Routing: GoRouter
- Version Control: GitHub

## Core Principles
- Do not recreate the project.
- Build incrementally.
- Preserve existing functionality.
- Prefer reusable components.
- Separate UI, Services, Models and Data Layers.
- Mobile compatibility should remain possible.
